Pagewise options

Welcome to Pagewise

Three quick steps to get started:

  1. Step 1. Add your Anthropic API key.
  2. Step 2. Enable at least one domain.
  3. Step 3. Open a tab on an enabled domain, press Alt+Shift+P, then click Run.

Enabled domains

Pagewise is inert on every web page until you enable it for that domain. Once enabled, Pagewise can extract page structure on that domain when you open the side panel. The page is sent to Anthropic only when you explicitly request an AI-enriched view.

Enter a hostname like example.com. You can paste a full URL — Pagewise will use just the domain.

Debug logs

Off by default. When on, Pagewise writes structured log entries to chrome.storage.session (cleared on browser restart). API keys, Authorization headers, and long string runs are redacted before storage. Inspect via Chrome DevTools → Application → Storage when you need to file a bug.

Cost disclosure
